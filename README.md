# SuhbatAI — Frontend Developer Interview Chat

## Project Overview
**SuhbatAI** is an interactive, AI-inspired interview preparation tool designed to help candidates practice their answers for the **Frontend Developer** role. Users can simulate a real interview by answering profession-specific questions in a chat-like interface.

The project currently uses a **predefined question bank** for frontend skills, including HTML, CSS, JavaScript, frameworks, performance, and accessibility. The chat interface provides a realistic, engaging, and structured interview experience.

---

## Features
- Profession-specific interview questions (Frontend Developer).  
- Interactive chat interface with user and AI messages.  
- Predefined question bank to cover critical frontend skills.  
- Tracks the number of questions answered (6 questions per interview).  
- Displays interview completion status.  
- Responsive and modern design for mobile and desktop.  
- Easy to customize questions via JSON in `chat.js`.
